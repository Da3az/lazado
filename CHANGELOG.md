# Changelog

## [0.1.0] - 2026-03-01

### Added
- Initial release
- Interactive fzf-based navigator (`ado` command)
- Work item management: view, create, search, update status
- User story management: view, create, search, update status
- Pull request management: list, create, approve, complete, checkout
- Pipeline run listing
- Repository listing, cloning, browser open
- Configuration system with global and project-level config files
- Dynamic work item state discovery via Azure DevOps REST API
- Auto-detection of organization and project from git remote
- Setup wizard (`ado-init`)
- Cross-platform browser support (Linux, macOS)
